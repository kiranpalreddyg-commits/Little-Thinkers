Orchestrator Agent
Master controller · Layer 1
Triggers
User session open · Cron 00:00 · Parent request
Classifies
daily_content | user_action | parent_report
Manages
Parallel agent spawning · Timeout handling · Retry policy
Output
RoutingContext object → downstream agents

Personalisation Agent
Context builder · Layer 2
Reads
PlayerProfile · readHistory · sessionLogs · streak
Computes
ageVariant · topicWeights · difficultyBand · moodHint
Decision
Profile complete? → proceed | → onboarding flow
Output
PersonalisedContext → all 3 generator agents
📋
Content Curator Agent
Selection logic · Layer 3
Checks
Redis/KV cache for today's pre-built slot
Decision
Cached? → skip generation · Not cached? → spawn 3 generators
Avoids
Topics/stories seen in last 30 days
Output
ContentRequest × 3 → Why/Story/Puzzle agents

Why Generator Agent
Curiosity content · Layer 4
Source
content/tell-me-why/ JSON packs
LLM call
Adapt text to ageGroup · Generate keyTerms · Rephrase nextQuestion
Validates
Flesch-Kincaid readability · Factual heuristic check
Output
WhyTopic payload → Safety Agent
📖
Story Generator Agent
Narrative content · Layer 4
Source
content/story-time/ JSON packs
LLM call
Select story by theme · Adapt reading level · Generate moral quiz (10+)
Validates
Word count 400–600 · Moral presence check · Age tag match
Output
Story payload → Safety Agent
🧩
Puzzle Selector Agent
Game content · Layer 4
Logic
Rotate across 5 games · Weight by least-played
Difficulty
Streak ≥ 7 → +1 difficulty · Streak broken → ease back
Attaches
Reset timer · Spark reward amounts · World area metadata
Output
DailyPuzzle payload → Safety Agent
🛡
Safety & Moderation Agent
Mandatory gate · Layer 5
Checks
Toxicity score · Age-appropriateness · COPPA flags · Readability band
Threshold
Score ≥ 0.92 to pass. Below → regenerate (max 3×)
Fallback
After 3 failures → serve pre-vetted static content
Logs
All failures to moderation audit trail
⭐
Reward Engine Agent
Incentive logic · Layer 6
Triggers
Session end · Read complete · Reaction tap · Day change
Calculates
sparkDelta · streakUpdate · badgeConditions · jarFillPercent
BadgeEval
Evaluates all BadgeDefinitions against session + cumulative data
Output
RewardTransaction + EarnedBadge[] → Data Persistence
📊
Parent Report Agent
Insight generation · Layer 8
Fetches
7-day session log · badge log · streak history · Why/Story reads
Generates
BrainReport · SkillRadar · topThinkingStyle · certificate flag
Outputs
PDF-ready report · Push notification copy · Leaderboard delta
Updates
PersonalisedContext via weekly learning loop (loop ③)
💾
Data Persistence Agent
Single API gateway · Layers 2 + 7
Rule
ONLY agent that touches the REST API. No other agent calls POST/GET directly.
Reads
GET /player · GET /sessions · GET /progress · GET /daily-content
Writes
POST /session · POST /rewards · POST /badges · PATCH /player
Caches
Daily content in KV store (Redis/Upstash) for instant loads
